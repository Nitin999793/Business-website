\# TasteBite Restaurant Website



Project Overview

TasteBite is a responsive, multi-page restaurant website built using HTML, CSS, SASS, and JavaScript.  

The website showcases restaurant services, food items, and contact information with a clean UI.







&nbsp;Features

\- Responsive design (mobile, tablet, desktop)

\- Mobile hamburger menu toggle

\- CSS Grid \& Flexbox layouts

\- Image gallery for food items

\- Contact form with validation

\- Accessibility-friendly structure







&nbsp;Technologies Used

\- HTML5

\- CSS3 (Grid \& Flexbox)

\- SASS / SCSS

\- JavaScript (Vanilla JS)
